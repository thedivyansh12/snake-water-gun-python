#  Snake Water Gun Game (Python)

This is a **simple Snake Water Gun game** built using Python.  
The project was created while learning **Python basics**, focusing on logic building and conditional statements.

---

##  Game Rules

- **Snake drinks Water → Snake wins**
- **Water douses Gun → Water wins**
- **Gun kills Snake → Gun wins**
- Same choices result in a **draw**

---

##  Concepts Used

- Python `random` module  
- Dictionaries  
- Conditional statements (`if`, `elif`, `else`)  
- User input handling
